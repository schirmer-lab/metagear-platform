    <span class="mg-eyebrow"><span class="mg-section-num">01</span>MetaGEAR Explorer · Web portal</span>
    <h2 class="mg-feature__title">

    </h2>
    <p class="mg-feature__lead">
      Bring a sequence, a Pfam domain, or an MSP identifier. Get back where

      <li><strong>Explore MSPs</strong> — curated Metagenomic Species Pangenomes.</li>
      <li><strong>Filter by cohort</strong>: IBD, CRC, healthy controls. Every hit traces back to its sample.</li>

  <div class="mg-feature__copy">
    <span class="mg-eyebrow"><span class="mg-section-num">02</span>MetaGEAR Workflows · Pipeline</span>
    <h2 class="mg-feature__title">
      One Nextflow pipeline, <em>four cohesive workflows.</em>
    </h2>
    <p class="mg-feature__lead">
      Built on nf-core conventions. Take paired-end FASTQ files through quality
      control, taxonomic and functional profiling, and gene-centric analysis —
      reproducibly, on a laptop or an HPC cluster.
    </p>
    <ul class="mg-feature__points">
      <li><strong>QC &amp; decontamination</strong> with Kneaddata and TrimGalore.</li>
      <li><strong>Taxonomic profiling</strong> with MetaPhlAn.</li>
      <li><strong>Functional profiling</strong> with HUMAnN.</li>
      <li><strong>Gene-centric analysis</strong>, with optional contig catalog assembly.</li>
    </ul>
    <div class="mg-feature__meta">

{/* ===================== FEATURE 3 · TOOLS (CLI) ===================== */}
<section class="mg-feature mg-reveal">
  <div class="mg-feature__copy">
    <span class="mg-eyebrow"><span class="mg-section-num">03</span>MetaGEAR Tools · CLI</span>
    <h2 class="mg-feature__title">
      Install the pipeline. <em>Run a workflow.</em> Skip the Nextflow learning curve.
    </h2>
    <p class="mg-feature__lead">
      MetaGEAR Tools is a tiny command-line wrapper that installs the
      pipeline, detects your hardware, manages reference databases, and runs
      workflows with one command.
    </p>
    <ul class="mg-feature__points">
      <li><strong>Hardware-aware</strong>: sets resource limits to ~80% of available CPUs and RAM.</li>
      <li><strong>Version-locked</strong>: installs the matching pipeline release.</li>
      <li><strong>One install command</strong>, one run command per workflow.</li>
    </ul>
    <div class="mg-actions">
      <a class="mg-btn mg-btn--primary" href="/tools/">Install MetaGEAR Tools →</a>
      <a class="mg-btn mg-btn--link" href="https://github.com/schirmer-lab/metagear-tools">View source ↗</a>
